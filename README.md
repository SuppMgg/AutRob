# Автономный робот с алгоритмом локальной навигации

# Описание проекта
Проект представляет собой программно-аппаратный прототип автономного наземного робота, способного ориентироваться на местности без использования систем спутниковой навигации. Разработка актуальна для приграничных регионов РФ (Севастополь, Крым), где спутниковые сигналы подавляются.

# Реализованный алгоритм
Реализован модифицированный алгоритм A для построения кратчайшего пути на сетке 5×5. В отличие от классической версии:
- Введены метки очереди (1–4) для запоминания развилок и переключения на запасной маршрут при изначально неточном выборе вершины для перехода (более актуально для карты с большим количеством вершин)
- Карта препятствий динамически обновляется (стены можно как добавлять, так и удалять)

# Аппаратная платформа
- Микроконтроллер: DXL IoT Board (Atmega2560)
- Датчики расстояния: 3×VL53L0X
- Инерциальный модуль: MPU6050 (IMU, 6 DOF)
- Приводы: 2 сервопривода Dynamixel AX-12W (TTL шина)
- Питание: Li-Po 2S-3S → DC-DC 5 В

# Программная реализация
- Язык программирования: C++ (Arduino IDE)
- Ключевые функции:
  - wave() — волновой алгоритм для построения эвристики
  - searchPath() — модифицированный A* (виртуальное планирование пути)
  - newPlace() — выбор реального направления движения, обновление карты при обнаружении различий между текущими данными с дальномеров и карты препятствий
  - moveTo() — исполнение движения
